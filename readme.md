# Team Finder

**Team Finder** — это веб-платформа для объединения IT-специалистов. Сервис помогает авторам идей находить разработчиков, дизайнеров и менеджеров в свои стартапы, а специалистам — находить интересные проекты для прокачки навыков и пополнения портфолио.

## Возможности проекта

- **Профили пользователей:** Регистрация, авторизация, настройка личного профиля (аватарка, контакты, ссылки на GitHub, раздел "О себе").
- **Управление проектами:** Создание карточек проектов с описанием, управление статусом (Открыт/Закрыт).
- **Система навыков (Skills):** Динамическое добавление необходимых стеков технологий к проекту (реализовано через AJAX без перезагрузки страницы).
- **Поиск и фильтрация:** Удобная фильтрация списка проектов по требуемым навыкам (тегам) и текстовый поиск.
- **Взаимодействие:** Возможность присоединиться к команде проекта или добавить его в закладки (Избранное).
- **Личный кабинет:** Удобное разделение проектов в профиле на вкладки: "Мои проекты", "Участвую" и "Избранное".

## Стек технологий

- **Backend:** Python 3.10, Django 5.2.4
- **База данных:** PostgreSQL 16 (psycopg2-binary)
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla/AJAX)
- **Инфраструктура:** Docker, Docker Compose
- **Управление окружением:** python-decouple

## Переменные окружения (.env)

Для безопасного запуска проекта необходимо создать файл `.env` в корневой директории (там же, где находится `manage.py`). 

Пример заполнения файла `.env`:

```env
# Базовые настройки Django
DJANGO_SECRET_KEY=ваш_надежный_секретный_ключ_здесь
DJANGO_DEBUG=True
ALLOWED_HOSTS=127.0.0.1,localhost,web

# Настройки подключения к базе данных PostgreSQL (для Docker)
POSTGRES_DB=team_finder
POSTGRES_USER=team_finder
POSTGRES_PASSWORD=team_finder
POSTGRES_HOST=db
POSTGRES_PORT=5432
```
*Примечание: Так как проект разворачивается в Docker, `POSTGRES_HOST` должен быть строго равен `db` (имени сервиса базы данных).*

## Запуск проекта (Docker)

Проект полностью контейнеризирован. Для его запуска на вашем компьютере должен быть установлен [Docker](https://www.docker.com/) и Docker Compose.

1. Склонируйте репозиторий и перейдите в папку проекта:
   ```bash
   git clone <ссылка_на_ваш_репозиторий>
   cd team-finder
   ```

2. Создайте файл `.env` и заполните его по примеру выше.

3. Запустите сборку и старт контейнеров:
   ```bash
   docker-compose up --build
   ```
   *При старте контейнера автоматически применятся все необходимые миграции базы данных.*

4. Создайте суперпользователя (администратора) для доступа к панели управления:
   ```bash
   docker-compose exec web python manage.py createsuperuser
   ```

5. Проект будет доступен в браузере по адресу:  
   **http://127.0.0.1:8000/**
